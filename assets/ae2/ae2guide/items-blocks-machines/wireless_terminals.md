# 无线终端

<ItemImage id="wireless_terminal" scale="4" />


无线终端是有线[终端](terminals.md)的便携版本。它们的界面与有线版本完全相同，不同之处在于它们没有<ItemLink id="view_cell" />的插槽，取而代之的是[升级卡](upgrade_cards.md)的插槽。

要将无线终端与网络配对，请将终端插入连接到该网络的<ItemLink id="wireless_access_point" />右上角的插槽中（带有无线终端图标和下方箭头的那个插槽）。

无线终端必须在<ItemLink id="wireless_access_point" />的范围内才能正常工作。

它们的能量可以在<ItemLink id="charger" />中充能。
